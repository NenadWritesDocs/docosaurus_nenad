Style guide rules for writing

| Use           | Do Not Use    |
|---------------|---------------|
| Active voice  | Passive voice |
| Short sentences | Long, complex sentences |
| Consistent terminology | Synonyms for the same concept |
| Present tense | Future or past tense (unless necessary) |
| Clear headings | Vague or missing headings|
|You|User|
|Sentence case for titles|Title case for titles|
